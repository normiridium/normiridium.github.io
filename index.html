Привет. Здесь живут код, архитектура и упрямое желание делать сложное понятным.

Разработка здесь — не про шум вокруг технологий, а про ясность мышления, надежность решений и интерес к системам, которые должны работать долго, честно и предсказуемо.

Коротко

  • ФокусBackend / Fullstack
  • Основной языкGo
  • ДополнительноTypeScript, React, Angular, SQL
  • ПодходDRY / KISS / SOLID + здравый смысл
  • Интересархитектура, производительность, безопасность

О подходе

В основе работы — системное мышление: разложить сложную задачу на понятные части, убрать лишнее, сохранить устойчивость и не потерять человеческий смысл за технической реализацией.

Особенно интересны backend-разработка, конкурентность, проектирование API, тестовые инструменты, документогенерация, инфраструктурные мелочи, которые обычно считают второстепенными, но именно они делают систему действительно живой и пригодной для долгой поддержки.

Рядом с кодом всегда идут внимание к UX, наблюдаемости, приватности и тому, как человек реально взаимодействует с интерфейсом, процессом или продуктом.

Стек

Go TypeScript JavaScript React Angular PostgreSQL MS SQL MongoDB RabbitMQ Docker REST API Swagger / OpenAPI

Что важно в работе

Надежность вместо магии Идемпотентность, отказоустойчивость, предсказуемое поведение, аккуратная работа с конкурентностью.
Читаемость как часть архитектуры Код должен не просто выполняться, а оставаться понятным спустя месяцы и годы.
Производительность без культа микрооптимизаций Оптимизация там, где она действительно влияет на поведение системы и опыт использования.
Инструменты для людей Документация, тестовые утилиты, шаблоны и внутренние сервисы важны не меньше публичных функций.
